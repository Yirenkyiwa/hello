    • Welcome to global code summer programme!!

    Global Code Summer Programme is a programmme organized for students in the various tertiary institutions across the world including Ghana. It is organised by some companies abroad who have partnered and have reached an agreement to help students who are curious to learn more about programming and electronics. Global Code focuses on teaching a programming language called Python and the use of raspberry pi to programme and do other stuff. The founder of Global Code is Sam Moorhouse, a very intelligent man. Students are given the opportunity to interact freely with the tutors on lessons that they don't understand very well and are free to ask their tutors any question. During the last week of the Summer Programme, students are given the opportunity to build projects of their own to demonstrate what they have learnt so far from the programme and it's a very nice experience. Global Code is all about fun!!! fun!!! fun!!!
